# SonixMac v1.0.0 Release Notes

Welcome to the initial stable release of **SonixMac (v1.0.0)**!

## New Features & Highlights
- **App Icon Implementation**: Added a new transparent high-resolution app icon.
- **In-App Version Display**: You can now view the current app version natively within the Settings View.
- **Automated Versioning**: Implemented a GitHub Action workflow to automatically bump the app's version on new code pushes to the `main` branch.
- **Audio Processing**: Start managing your local and remote backend connections directly from the UI.
- **Settings Configuration**: Easily configure concurrent processing limits and backend URL configurations.

## Setup Instructions
To run this release locally:
1. Download the source code.
2. Run `./build_app.sh` in the terminal to compile the release binary and bundle `Sonix.app`.
3. Open `Sonix.app` or move it to your Applications folder.

Enjoy the new streamlined macOS experience!
